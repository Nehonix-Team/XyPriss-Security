## EncryptionService

A production-grade encryption utility with automated salt management and binary sanitization.

### Methods

#### static async encrypt(data, key, options?)

Encrypts serializable data into a versioned JSON package.

**Parameters:**

- `data`: `any` - The data to encrypt.
- `key`: `string` - The master passphrase (minimum 8 characters).
- `options`: `EncryptionOptions` (Optional)
  - `algorithm`: `"aes-256-gcm" | "chacha20-poly1305"` (Default: `"aes-256-gcm"`)
  - `keyDerivationIterations`: `number` (Default: 100,000)
  - `quantumSafe`: `boolean` - If true, forced to ChaCha20-Poly1305.

**Returns:** `Promise<string>` (Stringified JSON package)

**Example:**

```typescript
import { EncryptionService } from "xypriss-security";

const secret = await EncryptionService.encrypt({ pin: 1234 }, "my-safe-key");
```

#### static async decrypt(encryptedData, key)

Decrypts a package generated by `encrypt`.

**Parameters:**

- `encryptedData`: `string` - The JSON package string.
- `key`: `string` - The master passphrase used for encryption.

**Returns:** `Promise<any>`

---

#### static generateSessionKey()

Generates a secure 256-bit session key in hexadecimal format.

**Returns:** `string`

#### static verifyIntegrity(encryptedData)

Checks if the encrypted package format is valid without attempting decryption.

**Parameters:**

- `encryptedData`: `string`

**Returns:** `boolean`

#### static getMetadata(encryptedData)

Extracts version and algorithm info from the package.

**Parameters:**

- `encryptedData`: `string`

**Returns:** `{ algorithm: string, timestamp: number, version: string }`
